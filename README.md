# T71 - AI Powered Smart Recycling Assistant (Low-Cost IoT + OCR)

## Team Members
- Parv Upadhyay (Team Lead) - Mobile App & Backend
- Vanshika Goyal - AI Engine & Web Platform  
- Sankalp Gupta - AI Models & Knowledge Base

## Project Structure
- **mobile_app/** - Flutter application with Firebase integration
- **ai_engine/** - TFLite waste classification models and training notebooks
- **backend_api/** - REST API services and disposal logic
- **web_platform/** - Next.js web dashboard
- **knowledge_base/** - Location-aware waste disposal rules

## Tech Stack
Flutter | Python | TensorFlow | Firebase | Next.js | OCR | IoT
